# 📊 Machine Learning Regression Practice

This project contains hands-on implementation of various **Regression Algorithms** using Python and Scikit-learn. It is part of my journey to learn and practice Machine Learning concepts from scratch.

---

## 🚀 Algorithms Covered

### 1. Simple Linear Regression

* Predicts a target variable using a single feature
* Example: Salary vs Experience

### 2. Multiple Linear Regression

* Uses multiple features to predict output
* Example: House Price Prediction

### 3. Polynomial Regression

* Handles non-linear relationships
* Transforms features into polynomial terms

### 4. Ridge Regression (L2 Regularization)

* Reduces overfitting by shrinking coefficients
* Penalizes large weights

### 5. Lasso Regression (L1 Regularization)

* Performs feature selection
* Some coefficients become zero

### 6. ElasticNet Regression

* Combination of Ridge and Lasso
